Documentation

About the Component

Add the custom CSS class pace-btn in combination with one of the following classes to achieve the appropriate PACE button styling.

  • pace-btn-primary
  • pace-btn-secondary
  • pace-btn-default

Add the custom CSS class pace-icon for basic icon styling.


Best Practices

Primary and secondary buttons are to be decided on a per wizard basis. Primary buttons are used for key calls to action, such as [Search] and [Submit]. Secondary buttons are used for optional calls to action, such as [Reset] and [Export].
