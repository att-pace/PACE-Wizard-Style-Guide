PACE Wizard Style Guide
2.6.2017


Welcome to the living style guide for PACE Wizards! Here you'll find reusable UI components that can be implemented across all PACE Wizards in the trainsition to HTML5.

The style guide utilizes Bootstrap as its base front-end framework, taking advantage of its intuitive and mobile-first functionality. Applicable Bootstrap components, such as navigation and modals, are overrided with the appropriate PACE styling.

Bootstrap is not required as the front-end framework, but was chosen due to it currently being the most popular and industry standard. PACE Wizard Style Guide components are usable with or without Bootstrap.

Font Awesome is a supplementary font and CSS toolkit that provides a wide range of scalable vector icons. The style guide uses Font Awesome icons for various buttons, such as 'Edit', 'Copy', and 'Delete'.
