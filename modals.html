Documentation

About the Component

Modals are built off of Bootstrap's modal. Use the custom class pace-modal to achieve the appropriate PACE modal styling.


Best Practices

Use a modal popup when the user should complete an action one at a time. This ensures that the user cannot use the main interface while the modal popup is in view.
