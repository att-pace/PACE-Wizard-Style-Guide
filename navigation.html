    Documentation

    About the Component

    The navigation bar is built off of Bootstrap classes navbar and navbar-default. Responsive functionality is maintained, scaling appropriately from desktop to mobile.

    To achieve the appropriate PACE navbar styling, add the custom CSS class pace-navbar to the navigation.


    Best Practices

    The navigation bar is displayed in every PACE wizard. The name in the top left will be unique per wizard, but otherwise the navbar is consistent across wizards.
