Documentation

About the Component

Wrap tables in the pace-table-container and add the custom class pace-table to tables in order to apply PACE table styling.

Use the custom background-color CSS classes to easily style table headings, i.e. pace-bg-base

To desigate a button, form control, or table element is disabled:
  • Add the disabled attribute to the respective element.

To designate that a table cell/row has been edited:
  • Add the has-edit custom class to the edited table row
  • Add the fa-pencil Font Awesome icon to to the edited table cell and row
