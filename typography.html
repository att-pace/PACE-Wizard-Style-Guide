Documentation

About the Component

Aleck is AT&T's custom font and is used for all heading text.
Body text utilizes Source Sans Pro, a modern sans-serif font.

The following custom classes may be used to modify text coloring:

  • pace-text-primary
  • pace-text-primary-darker
  • pace-text-primary-darkest
  • pace-text-white
  • pace-text-gray
  • pace-text-base

The following custom classes may be used to modify text sizing:

  • pace-text-xs
  • pace-text-sm
  • pace-text-md
  • pace-text-lg
  • pace-text-xl

The following helper classes may be used to achieve a particular text styling or layout:

  • text-center
  • text-right
  • text-left
  • bold
  • italic
  • underline
  • lowercase
  • uppercase
  • capitalize
